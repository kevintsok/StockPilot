v0.0.2: Strategy Abstraction & Multi-Strategy Backtest

## What's New

### Strategy Architecture
- BaseStrategy ABC + Signal dataclass for pluggable strategies
- 10 pre-defined strategies in JSON format (strategies/configs/default_strategies.json)
- StrategyRegistry loads and validates JSON configs with jsonschema

### Shared Signal Collection
- run_all_strategies_shared(): one GPU pass for all stocks/dates
- Each strategy independently calls select_positions() with its own cache
- ~10x faster than running each strategy through the full pipeline separately

### New CLI Command
- backtest-strategies --list: preview all available strategies
- backtest-strategies --start/--end: run all strategies and compare results

## 10 Strategies Included
1. TopK-Proportional - baseline proportional weighting
2. TopK-Threshold - only buy when pred > 1%
3. TopK-StopLoss - with trailing stop (fails on A-share T+1)
4. LongShort-Equal - long top 10%, short bottom 10%
5. Momentum-Filter - buy top-K only when pred trending up
6. Risk-Parity - inverse-vol weighting among top-K
7. BottomK-Reversal - long losers, short winners
8. Confidence-Sized - weight by |pred_ret| (BEST: +71.1% net)
9. Sector-Neutral - net-zero sector exposure
10. Dual-Threshold - long above 2%, short below -1%
